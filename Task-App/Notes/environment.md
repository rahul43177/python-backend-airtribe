## Virtual Environment
- A virtual environment is a self-contained directory that contains all the necessary files and dependencies for a specific project. It allows you to manage and isolate the dependencies of different projects, preventing conflicts between them.

We can create a venv for the project specific and install it globally as well for entire system.
